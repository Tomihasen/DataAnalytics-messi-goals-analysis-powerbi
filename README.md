# Lionel Messi Goals Analysis – Power BI

## Project Overview

This project analyzes all goals scored by Lionel Messi at FC Barcelona and PSG. 
The objective is to explore scoring trends, performance patterns, and goal distribution using an interactive Power BI dashboard.

## Tools Used

- Power BI
- DAX
- Power Query
- CSV dataset

## Key KPIs

- Total goals
- Goals per season
- Goals per competition
- Goals by minute range
- Goals by club
